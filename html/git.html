Git

  • ls - расшифровывается как list (список). Она используется для вывода списка файлов и каталогов в текущей директории или в указанной директории.
  • cd - расшифровывается как "Change Directory", что переводится как "сменить каталог" или "изменить директорию". Эта команда используется для навигации по файловой системе, то есть для перемещения между папками и каталогами.
  • pwd - расшифровывается как "print working directory" (вывести текущий рабочий каталог). Она отображает полный путь к текущему каталогу, в котором вы находитесь в терминале или командной строке.
  • mkdir - расшифровывается как «make directory", что в переводе с английского означает "создать каталог". Эта команда используется для создания новых папок или каталогов в файловой системе, а не в Git непосредственно.
  • mv - расфшифровывается как "move" и используется для перемещения или переименования файлов и директорий в репозитории Git.
