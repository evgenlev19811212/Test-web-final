Git

  • git clone - привязка удаленного репозитория к локальному репозиторию.
  • git commit-m - создание коммита, сохранение изменений.
  • git push - отправка изменений на удаленный репозиторий.
  • git log - проверка изменений, коммитов, какие команды совершали.
  • git pull - перенос изменений из локального репозитория на удаленный репозиторий.
  • git status - показывает текущее состояние репозитория, включая изменения в рабочих файлах, а также файлы, ожидающие коммита.
  • git cls - очистить терминал.
